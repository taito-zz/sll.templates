Log in as the portal owner::

    >>> from plone.app.testing import TEST_USER_NAME, TEST_USER_PASSWORD
    >>> browser.login(TEST_USER_NAME, TEST_USER_PASSWORD)

Go to document sll-view::

    >>> browser.open('/document')

    >>> '@@feed-to-top' in browser.contents
    True
    >>> '@@unfeed-from-top' in browser.contents
    False

    >>> browser.getLink('Feed to Top').url
    'http://nohost/plone/document/@@feed-to-top'

    >>> browser.getLink('Feed to Top').click()
    >>> '@@feet-to-top' in browser.contents
    False

    >>> '@@feed-to-top' in browser.contents
    False
    >>> '@@unfeed-from-top' in browser.contents
    True

    >>> browser.getLink('Unfeed from Top').url
    'http://nohost/plone/document/@@unfeed-from-top'

    >>> browser.getLink('Unfeed from Top').click()

    >>> '@@feed-to-top' in browser.contents
    True
    >>> '@@unfeed-from-top' in browser.contents
    False
